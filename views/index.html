webhook2status

webhook2status: an awesome and simple request generator to change Slack status via webhook, IFTTT and curl. / View on GitHub

Login with Slack

Install / Usage

  1. Login with Slack
  2. Get an API token through OAuth login
  3. Use the token whatever from IFTTT, curl, etc.
  4. To create API request payload including own status message, this API tester in Slack documentation is very useful

Grant Permission

This tool issues following permission scopes:

  • user.profile:write

Uninstall / Revoke API access

Go to the Slack App manage page and revoke webhook2status.

Notice

This service does not store ANY sensitive data including your persistent OAuth token, password or access log.
